    📖 사용법 안내
    
                
            
    
            
    
                
    
                    
    🎯 주요 기능
    
                    
      
                        
    • 그리드 시스템 계산: 컨테이너 너비, 컬럼 수, 간격, 마진을 입력하면 자동으로 컬럼 너비를 계산합니다.
      • 
                        
    • 가이드라인 생성: 피그마에서 사용할 가이드라인 위치를 정확하게 계산합니다.
      • 
                        
    • 시각화: 계산된 그리드 시스템을 시각적으로 확인할 수 있습니다.
      • 
                        
    • CSS 코드 생성: CSS Grid나 Flexbox로 구현할 수 있는 코드를 생성합니다.
      • 
                    
    
                
    
                
    
                    
    📝 사용 방법
    
                    
      
                        
    1. 그리드 설정 입력:
                            
        
                                
      • 컨테이너 너비: 전체 레이아웃의 너비를 입력합니다 (예: 1440px, 1920px)
        • 
                                
      • 컬럼 수: 그리드의 컬럼 개수를 입력합니다 (예: 12, 16)
        • 
                                
      • 간격 (Gutter): 컬럼 사이의 간격을 입력합니다 (예: 24px)
        • 
                                
      • 마진 (Margin): 좌우 마진을 각각 입력합니다 (예: 80px)
        • 
                            
      
                        
      2. 
                        
    2. 계산 결과 확인:
                            
        
                                
      • 컬럼 너비, 사용 가능 너비 등이 자동으로 계산되어 표시됩니다.
        • 
                            
      
                        
      3. 
                        
    3. 가이드라인 사용:
                            
        
                                
      • 피그마에서 사용할 가이드라인 위치가 표시됩니다.
        • 
                                
      • "가이드라인 복사" 버튼을 클릭하여 위치 값을 복사할 수 있습니다.
        • 
                            
      
                        
      4. 
                        
    4. CSS 코드 사용:
                            
        
                                
      • 생성된 CSS 코드를 복사하여 프로젝트에 적용할 수 있습니다.
        • 
                                
      • "CSS 코드 복사" 버튼을 클릭하여 코드를 복사합니다.
        • 
                            
      
                        
      5. 
                    
    
                
    
                
    
                    
    💡 계산 공식
    
                    
      
                        
    • 사용 가능 너비: 컨테이너 너비 - (마진 × 2)
      • 
                        
    • 총 간격 너비: 간격 × (컬럼 수 - 1)
      • 
                        
    • 컬럼 너비: (사용 가능 너비 - 총 간격 너비) ÷ 컬럼 수
      • 
                    
    
                
    
                
    
                    
    📋 피그마 사용 팁
    
                    
      
                        
    • 피그마에서 Layout Grid를 추가하고 계산된 값을 입력하세요.
      • 
                        
    • 가이드라인 위치 값을 참고하여 정확한 레이아웃을 구성하세요.
      • 
                        
    • 컬럼 수와 간격은 디자인 시스템에 따라 조정하세요.
      • 
                        
    • 반응형 디자인을 고려하여 여러 브레이크포인트별로 계산하세요.
