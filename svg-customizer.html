🎯 주요 기능

                    
    
                        
  • SVG 파일 업로드: SVG 파일을 업로드하여 편집할 수 있습니다.
    • 
                        
  • SVG 코드 입력: SVG 코드를 직접 입력하거나 붙여넣을 수 있습니다.
    • 
                        
  • 색상 변경: SVG 아이콘의 색상을 변경할 수 있습니다.
    • 
                        
  • 크기 조절: 너비와 높이를 조절할 수 있습니다.
    • 
                        
  • 회전: SVG 아이콘을 회전시킬 수 있습니다.
    • 
                        
  • 다운로드: 편집된 SVG 파일을 다운로드할 수 있습니다.
    • 
                    

                

                

                    
📝 사용 방법

                    
    
                        
  1. SVG 업로드:
                            
      
                                
    • 파일 선택 버튼을 클릭하여 SVG 파일을 업로드하거나
      • 
                                
    • "SVG 코드 붙여넣기" 버튼을 클릭하여 SVG 코드를 입력합니다.
      • 
                            
    
                        
    2. 
                        
  2. 편집 옵션 설정:
                            
      
                                
    • 색상: 색상 피커나 HEX 코드로 색상을 변경합니다.
      • 
                                
    • 크기: 너비와 높이를 픽셀 단위로 조절합니다.
      • 
                                
    • 회전: 슬라이더를 사용하여 회전 각도를 조절합니다.
      • 
                                
    • 비율 유지: 체크박스를 선택하면 너비와 높이 비율이 유지됩니다.
      • 
                            
    
                        
    3. 
                        
  3. 미리보기 확인: 변경 사항을 실시간으로 확인합니다.
    4. 
                        
  4. 다운로드: "SVG 다운로드" 버튼을 클릭하여 편집된 SVG를 저장합니다.
    5. 
                    

                

                

                    
💡 팁

                    
    
                        
  • SVG 코드를 직접 편집하여 더 세밀한 조정이 가능합니다.
    • 
                        
  • 색상 변경은 SVG의 fill 속성을 변경합니다. stroke 속성이 있는 경우 stroke도 변경됩니다.
    • 
                        
  • 비율 유지를 선택하면 너비를 변경하면 높이가 자동으로 조절됩니다.
    • 
                        
  • 회전은 0도부터 360도까지 가능합니다.
    • 
                        
  • 일부 복잡한 SVG의 경우 색상 변경이 제대로 작동하지 않을 수 있습니다.
    •
