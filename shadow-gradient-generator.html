📖 사용법 안내

                
            

            

                

                    
🎯 주요 기능

                    
    
                        
  • 박스 그림자: 요소에 그림자 효과를 추가하는 CSS 코드를 생성합니다.
    • 
                        
  • 텍스트 그림자: 텍스트에 그림자 효과를 추가하는 CSS 코드를 생성합니다.
    • 
                        
  • 그라디언트: 선형 및 방사형 그라디언트 CSS 코드를 생성합니다.
    • 
                        
  • 실시간 미리보기: 설정을 변경하면 즉시 결과를 확인할 수 있습니다.
    • 
                        
  • CSS 코드 복사: 생성된 CSS 코드를 쉽게 복사하여 프로젝트에 사용할 수 있습니다.
    • 
                    

                

                

                    
📝 사용 방법

                    
    
                        
  1. 효과 타입 선택: 박스 그림자, 텍스트 그림자, 또는 그라디언트 중 하나를 선택합니다.
    2. 
                        
  2. 설정 조절:
                            
      
                                
    • 박스 그림자: X/Y 오프셋, 블러, 확산, 색상, 투명도를 조절합니다.
      • 
                                
    • 텍스트 그림자: X/Y 오프셋, 블러, 색상, 투명도를 조절합니다.
      • 
                                
    • 그라디언트: 타입(선형/방사형), 방향, 색상을 설정합니다.
      • 
                            
    
                        
    3. 
                        
  3. 미리보기 확인: 실시간으로 변경 사항을 확인합니다.
    4. 
                        
  4. CSS 코드 복사: "CSS 복사" 버튼을 클릭하여 코드를 복사합니다.
    5. 
                    

                

                

                    
💡 팁

                    
    
                        
  • 박스 그림자의 확산 값은 음수로 설정하면 그림자가 작아집니다.
    • 
                        
  • 텍스트 그림자는 여러 개를 중첩하여 사용할 수 있습니다.
    • 
                        
  • 그라디언트의 방향은 각도(deg)로도 설정할 수 있습니다.
    • 
                        
  • 색상 입력 필드에 HEX 코드를 직접 입력할 수 있습니다.
    • 
                        
  • 투명도는 0%부터 100%까지 조절할 수 있습니다.
    •
