🎯 주요 기능

                    
    
                        
  • 코드 검증: HTML, CSS, TypeScript 코드의 문법 오류를 검사합니다.
    • 
                        
  • 코드 포맷팅: 코드를 읽기 쉽게 정리하고 포맷팅합니다.
    • 
                        
  • 다중 언어 지원: HTML, CSS, TypeScript 세 가지 언어를 지원합니다.
    • 
                        
  • 오류 표시: 발견된 오류를 명확하게 표시하고 설명합니다.
    • 
                        
  • 실시간 검증: 코드를 입력하면 즉시 검증 결과를 확인할 수 있습니다.
    • 
                    

                

                

                    
📝 사용 방법

                    
    
                        
  1. 상단에서 코드 타입(HTML, CSS, TypeScript)을 선택합니다.
    2. 
                        
  2. 코드를 입력하거나 붙여넣습니다.
    3. 
                        
  3. "검증하기" 버튼을 클릭하여 코드를 검증합니다.
    4. 
                        
  4. "포맷팅" 버튼을 클릭하여 코드를 정리합니다.
    5. 
                        
  5. 검증 결과와 포맷팅된 코드를 확인합니다.
    6. 
                    

                

                

                    
⚠️ 주의사항

                    
    
                        
  • 일부 복잡한 코드는 완벽하게 검증되지 않을 수 있습니다.
    • 
                        
  • 외부 라이브러리나 프레임워크 코드는 검증이 제한적일 수 있습니다.
    • 
                        
  • 검증 결과는 참고용이며, 실제 프로덕션 환경에서는 추가 테스트가 필요합니다.
